# Line-Follower-Robot

It can the follow the black line with the help of sensor.

Technology Used:
  * Arduino IDE
  * C++ Programming
  
Components Used:
  * Arduino UNO R3
  * Ultrasonic Sensor
  * L293d Motor driver
  * DC Geared Motor
  * Lithium-ion Battery
  * Servo Motor
  * Jumper Wires
  * Infrared Sensor
